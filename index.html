This redesign project reimagines the Dallas Library's website with a focus on clarity, accessibility, and modern responsiveness. Starting with a usability audit, I identified key pain points and then used visual design principles, HTML/CSS implementation, and iterative testing to craft a more navigable and visually coherent alternative.

Part 1: Identifying Usability Problems

Chosen Interface

The Dallas Library website is publicly accessible and was selected due to its cluttered navigation and outdated visual elements that hinder efficient user interaction.

Usability & Accessibility Issues

  • Efficiency: The navigation menus and content layout are very cluttered, making it challenging for users to quickly find important features like event calendars or catalog search.
  • Learnability: Inconsistent visual cues and outdated design elements may confuse new users, affecting their ability to quickly learn the website's layout.
  • Memorability: A lack of consistent design patterns makes it difficult for returning users to remember where to locate key information.
  • Accessibility: A WebAIM WAVE analysis revealed issues like missing alt text, missing page regions, and very small text which could hinder users with visual impairments.

Project Reflection & Lessons Learned

This project was a powerful reminder that good design is often invisible: users notice it most when it's missing. Redesigning the Dallas Library website helped me think critically about accessibility not as a checklist, but as a mindset: designing for real users in real contexts.

By reworking layout hierarchies, visual cues, and responsive behavior, I created a more welcoming experience for both first-time visitors and longtime users. The process reinforced the value of style guides, mobile-first thinking, and cross-device testing as pillars of modern web design.

More broadly, this redesign emphasized the importance of advocacy through design. Simple, thoughtful changes can expand access to public resources and make a city's digital front door feel open to all.
