Responsive Redesign: Dallas Library Website

Introduction

For this project, I chose to redesign the Dallas Library website (dallaslibrary2.org) because its design appears dated and can benefit from a more modern, accessible, and responsive interface that better serves its diverse audience.

Part 1: Identifying Usability Problems

Chosen Interface

The Dallas Library website is publicly accessible and was selected due to its cluttered navigation and outdated visual elements that hinder efficient user interaction.

Usability & Accessibility Issues

  • Efficiency: The navigation menus and content layout are very cluttered, making it challenging for users to quickly find important features like event calendars or catalog search.
  • Learnability: Inconsistent visual cues and outdated design elements may confuse new users, affecting their ability to quickly learn the website's layout.
  • Memorability: A lack of consistent design patterns makes it difficult for returning users to remember where to locate key information.
  • Accessibility: A WebAIM WAVE analysis revealed issues like missing alt text, missing page regions, and very small text which could hinder users with visual impairments.

Project Reflection & Lessons Learned

This project allowed me to address significant usability and accessibility challenges on an existing public interface. By updating the visual style, simplifying navigation, and ensuring a responsive design, the redesigned Dallas Library website now provides a more intuitive and engaging user experience.

The process emphasized the importance of a cohesive style guide and cross-device testing, ensuring that design decisions enhance both user satisfaction and accessibility.
